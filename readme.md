# lumen2

这个仓库实现了一个用于优化微波光子滤波芯片运行参数的程序。

## 什么是光芯片的运行参数？

运行参数指的是芯片设计、制造完成后，实验者不改变芯片内部结构的情况下，在使用芯片时使用的电压、外部输入光强等参数。通过改变这些参数，芯片会呈现出不同的运行特性，例如，对于一个微波光子滤波器来说，可能会产生不同的响应曲线。

## 为什么要优化运行参数？怎么优化?

如同上一节所说，不同的运行参数会使得光芯片产生不同的运行特性，而我们希望取得符合我们期望的运行特性，例如，在我们期望的阻带具有较强的衰减，并且在通带几乎没有损耗。

不幸的是，这一过程在过去往往依赖于实验者手动的经验调节，因此这在实验中往往需要漫长的时间。因此，我们希望利用程序化的方法来优化这些运行参数。

一个重要的困难是在实验的频率范围非常狭窄时，高精度的光谱分析仪（OSA）读取响应曲线的速度非常缓慢，这使得直接利用这一信息优化非常困难且缓慢。因此，我们引入一个由实验者负责的仿真模型，并且利用实际从OSA中获取的实际信息进行校准，通过这种方式，我们可以快速的获取仿真波形并进行优化，仅在少数时候请求OSA进行读取（系统辨识）。

## 项目结构

我们的项目提供一个网页可视化界面，用户可以可视化地：
* 指定光芯片结构和参数
* 指定优化目标，例如上传一个滤波器的目标响应曲线
* 开始优化，并且查看优化进度和模型校准信息
* 下载保存校准后的模型
